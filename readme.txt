== Description ==

This plugin allows WordPress site administrators to customize their user order invoices by modifying the data displayed and choosing from multiple templates.

== Installation ==

- Download the plugin ZIP file.
- Upload the plugin to your WordPress site.
- Activate the plugin through the WordPress plugins page.
